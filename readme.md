## About The Project

A simple full stack application made using Node.js and MongoDB to add and delete events.  


![dashboard](doc/img/dashboard.png)



### Built With

- Nodejs
- Express
- MongoDB



### Prerequisites

`npm` `node` `mongo`




### Installation

- `npm i` or `yarn`
-  Create a `.env` file inside the app
-  Insert  `PORT = PORT_NUMBER ` `DB_NAME = YOUR_DB_NAME`
- `node server`
